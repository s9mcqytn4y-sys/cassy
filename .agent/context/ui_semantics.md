# UI Semantics

## Entry point
Guided Operations Dashboard is the operational control tower, not a generic home menu.

## Semantic states
- Info
- Success
- Warning
- Danger
- Blocked
- Pending Sync
- Approval Required
- Offline Restricted

## Stable meaning across platforms
These must remain semantically identical across Android POS, Desktop, and selective supervisor surfaces:
- severity language
- field naming
- critical CTA ordering
- gate semantics
- readiness meaning

## Native adaptation allowed
Navigation idiom, keyboard behavior, touch/pointer details, and layout density may adapt per platform.
