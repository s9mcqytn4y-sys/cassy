# Project Overview

Cassy V1 adalah desktop-first retail operating core untuk single outlet.

Fokus yang benar:
- guided operations yang tenang dan jelas
- cashier core yang usable
- inventory basic yang jujur
- approval, void, cash control, close shift/day yang explainable
- local-first snapshot dan sync queue yang tidak magic

Anti-scope:
- ERP breadth
- HQ multi-outlet analytics
- cloud-first assumptions
