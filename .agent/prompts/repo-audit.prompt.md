# Prompt: Repo Audit

"Gunakan `repo-audit-playbook.md` untuk melakukan audit pada repositori Cassy saat ini.

Fokus pada:
1. Konsistensi antara `shared/` modules dan `docs/cassy_architecture_specification_v1.md`.
2. Validasi apakah ada business logic yang bocor ke module `apps/`.
3. Pengecekan status migrasi SQLDelight.
4. Identifikasi 'orphan' modules atau files yang tidak digunakan.

Berikan output berupa laporan markdown yang terstruktur dengan section: Findings, Risks, and Recommendations."
