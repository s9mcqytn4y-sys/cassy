# Prompt: Gradle Task Audit

"Gunakan MCP Filesystem untuk membaca `settings.gradle.kts` dan `build.gradle.kts` di root serta modul-modul utama.

Tugas Anda:
1. Identifikasi task Gradle yang paling relevan untuk lifecycle:
   - Build (Assemble/Package)
   - Test (Unit Test/Instrumentation)
   - Database (SQLDelight Generation/Migration)
   - Release (Obfuscation/Signing)
2. Buat ringkasan dalam format tabel yang berisi: Task Name, Module, dan Purpose.
3. Rekomendasikan mana yang harus di-whitelist untuk MCP Terminal."
