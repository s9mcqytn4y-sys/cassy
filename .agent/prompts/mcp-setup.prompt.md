# Prompt: MCP Setup Checklist

"Buat panduan langkah demi langkah untuk developer tim Cassy yang ingin menyetel MCP di Android Studio / Gemini.

Pastikan panduan mencakup:
1. Prasyarat: Node.js, `npx`.
2. Link ke repository MCP server resmi (Filesystem, SQLite, Git).
3. Template JSON config untuk Windows (`C:\Users\Acer\AndroidStudioProjects\Cassy`).
4. Cara verifikasi apakah MCP sudah aktif dan terhubung."
